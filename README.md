# BookNest

A fully responsive online bookstore built using HTML, CSS, JavaScript, and Bootstrap. This project focuses on a user-friendly interface, clean design, and essential e-commerce features like Add to Cart, product listing, and smooth navigation.

## Features

- **Responsive Design:** Looks great on all devices, from desktops to mobiles.
- **Modern UI:** Clean and intuitive user interface for easy browsing.
- **Product Listing:** Display books with images, descriptions, and prices.
- **Add to Cart:** Easily add and manage products in your cart.
- **Smooth Navigation:** Effortless page transitions and product discovery.

## Built With

- HTML
- CSS
- JavaScript
- Bootstrap

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/nileshkaute/BookNest.git
   ```
2. Open the project folder.
3. Launch `index.html` in your preferred web browser.
